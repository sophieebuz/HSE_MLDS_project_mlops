Status of github actions:

Prerequisites before launch:
  - Python >=3.10
  - Poetry

Initial configeration:
  1. run `git clone git@github.com:sophieebuz/HSE_MLDS_project_mlops.git`
  2. run `poetry install`
  3. run `pre-commit install`
  4. run `dvc pull`
